Array对象方法

concat()

array1.concat(array2,array3,...,arrayX)
必需。该参数可以是具体的值，也可以是数组对象。可以是任意多个。

定义和用法： 用于连接两个或多个数组。
该方法不会改变现有的数组，而仅仅会返回被连接数组的一个副本。

quiz：传递两个参数m，n，返回长度为m，所有元素都为n的数组，要求不能用循环。

function fn(m, n) {
return m ? fn(m - 1, n).concat(n) : [];
}

copyWithin()

array.copyWithin(target, start, end)
target： 必需。将元素复制到的索引位置
start： 可选。开始复制元素的索引位置（默认为 0）。
end：可选。停止从中复制元素的索引位置（默认为 array.length）。

定义和用法： 将数组本身元素复制到数组本身中的另一个位置，覆盖现有值。
永远不会向数组添加更多项，但会覆盖原始数组。

entries()

array.entries()

定义和用法：返回带有键/值对的 Array Iterator 对象。
对于原始数组中的每一项，新的迭代对象将包含一个以索引为键，以项值为值的数组
不会更改原始数组。
Internet Explorer 不支持 ​entries()​ 方法。

例如： var fruits = ["Banana", "Orange", "Apple", "Mango"];
var f = fruits.entries();
for (x of f) {

fill()

array.fill(value, start, end)
value： 必需。用于填充数组的值。
start： 可选。开始填充数组的索引（默认为 0）。
end： 可选。停止填充数组的索引（默认为 array.length）。

定义和用法： 用静态值填充数组中的指定元素。指定开始和结束填充的位置。如果未指定，则将填充所有元素。
会覆盖原始数组。Internet Explorer 不支持 ​fill()​ 方法。

例如：var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.fill("Kiwi");

from()

Array.from(object, mapFunction, thisValue)
object： 必需。需转换为数组的对象。
mapFunction： 可选。对数组的每个项目调用的 map 函数。
thisValue： 可选。执行 mapFunction 时用作 this 的值。

定义和用法： 从具有 length 属性或可迭代对象的任何对象返回 Array 对象。
